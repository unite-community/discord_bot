# setup
1. invite using url (accept administrator permission)
2. role collab.land will be set up automatically - need to move this to top of roles list
3. add collab.land bot to all roles
4. go to collabland-config private channel set up when bot was added and type !setup
5. click emoji reactions in chat and navigate to setting up ERC20 role 
6. reply with parameters: token_address, min_tokens, max_tokens (-1 for unlimited), @roll_name
7. type !join to join token-permissioned channels
8. get DM with link
9. click link and go to webpage where you connect wallet
10. get DM saying roles updated (but doesn't tell you which roles changed)
